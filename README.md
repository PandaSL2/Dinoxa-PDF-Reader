# 🚀 Dinoxa PDF Reader & Editor (v1.0.2)

**Dinoxa** is a high-performance, Adobe-style desktop PDF workstation designed for speed, beauty, and intelligent document management. Built with a **Privacy-First** architecture, Dinoxa handles everything from multi-tab research to AI-powered document analysis directly on your machine with speed, stability, and modern design.

### 📥 [Dinoxa 1.0.2.exe](https://drive.google.com/file/d/1vuVO2NzndRUdn3XWYd70TF1jrAy-W3VH/view?usp=sharing)

---

## ✨ Key Features

* **⚡ Professional Multi-Tab Interface**: Open up to 10 PDF documents simultaneously. Swap between them instantly with a fluid, lag-free tab system designed for high-volume research.
* **🧠 Integrated AI Brain Hub**: A floating AI assistant that can summarize pages, extract key points, and answer complex questions about your documents in real-time.
* **🎨 Instant Annotations**: High-performance highlighting, underlining, and strike-through tools optimized with a dedicated rendering layer for smooth visual feedback.
* **🌗 Smart Dark Mode**: A custom-tuned dark theme that intelligently adapts document colors while preserving readability of images, charts, and diagrams.
* **📈 Advanced Rendering Engine**: Crystal-clear PDF rendering with a percentage-based zoom engine (up to 500%) and hardware-accelerated rendering for ultra-smooth navigation.
* **💾 Session Persistence**: Automatically resumes your reading exactly where you left off. The "Resume Reading" screen keeps your active documents one click away.
* **☁️ Secure Encrypted Cloud Sync**: Dinoxa securely stores synchronized document sessions using encrypted cloud technology. Even if a PDF is accidentally deleted or moved from your computer, previously synced documents and reading sessions can still be reopened seamlessly inside Dinoxa without interruption.
* **🔄 Intelligent Recovery System**: Automatically restores recently opened tabs, annotations, zoom positions, and workflow states after unexpected shutdowns or system restarts.

---

## 🛡️ Security & Privacy

* **🔐 End-to-End Protection**: Sensitive documents, annotations, and synced sessions are protected using secure encrypted storage technologies.
* **💻 Offline-First Architecture**: Core reading, annotation, and document management features work fully offline without requiring cloud access.
* **☁️ Privacy-Controlled Sync**: Cloud synchronization remains fully user-controlled. Users can permanently remove synced files and associated data at any time.
* **🧱 Hardened Application Security**: Built with isolated runtime processes and secure application architecture for stability, reliability, and tamper resistance.

## 🚀 How to Test

1. Download the **`Dinoxa PDF Reader_Setup_1.0.0.exe`** from the link above.
2. Run the installer on your Windows machine.
3. Launch **Dinoxa PDF Reader** from your desktop or start menu.
4. Drag and drop any PDF to experience premium rendering, annotations, and AI-powered tools.
